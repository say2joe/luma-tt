Thanks for your interest in joining the Luma Pictures dev team! This coding test will assess your comfort level with front-end web development. Although time is an important factor, it's more important that you write quality code. Feel free to improve aesthetics and usability as you see fit.

Your Tasks

You'll be working on making improvements to a simple task tracker. The tasks you'll need to complete are in the task list itself. Complete as much of the tasks as you can. Bonus points if you version your changes in a Git or Mercurial repository.

Submission

When you are finished, please zip your final project directory and submit it to chrisl@luma-pictures.com for review. Please also indicate what tasks you weren't able to complete. If you weren't able to complete some tasks, please briefly describe what you would have done to tackle the problem.

Testing

Jasmine UI tests are in the public/js/spec directory. If you're more comfortable with another testing framework (such as QUnit), feel free to use that instead. If you need to add new test spec files, you'll also need to edit the js/test.js file, so that they're included. You can access the Jasmine test runner at /test.html on this server.

Libraries

We've included some standard libraries for convenience. If you'd like to add more, feel free, but try to keep the amount of libraries used to a minimum. Here's a list of the included library documentation:

  • jQuery
  • jQuery UI
  • Bootstrap
  • Backbone.js
  • Underscore.js
  • RequireJS
  • Modernizr
  • Jasmine
  • Sinon
